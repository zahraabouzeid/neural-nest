**Function Definition**
```javascript
function add(x, y) {
  return x + y;
}
```

**Function Invocation**
```javascript
add(1, 2)
```

**Component Definition**
```javascript
function Icon () {
  return <svg />
}
```

**Element**
```javascript
<Icon />
```

We call it creating an element, because after the JSX is compiled and `jsx` is invoked, that’s what we get back, an element.
